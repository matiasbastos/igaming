- Install it::

	git clone https://github.com/matiasbastos/igaming.git
	cd igaming
	virtualenv env
	source env/bin/activate
	pip install -r requirements.txt

- Create admin user::

	fabmanager create-admin

- Run it::

	python run.py


That's it!!
